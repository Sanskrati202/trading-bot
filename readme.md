\# Trading Bot (Binance Futures Testnet)



\## Setup



1\. Install Python

2\. Install libraries:

&#x20;  pip install -r requirements.txt



\## How to Run



1\. Test connection:

&#x20;  python test.py



2\. Place order:

&#x20;  python order.py



\## Features



\- Place MARKET orders

\- Uses Binance Futures Testnet

\- Simple and easy to use



\## Note



\- API keys required

\- This is testnet (no real money)
